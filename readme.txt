== Description ==
A advance contact for widget.

== Installation ==
1. Upload the entire ks-contact-widget folder to the /wp-content/plugins/ directory
2. Activate the plugin through the \\\'Plugins\\\' menu in WordPress
3. In your Widgets menu, simply drag the widget labeled \\\"Ks Contact Widget\\\" into a widgetized Area.
4. Configure the widget by choosing a title, about me, link more, address, phone, twitter, icon size and color, and the URLs to your various social profiles.
